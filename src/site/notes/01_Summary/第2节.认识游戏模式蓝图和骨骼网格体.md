### 虚幻引擎基础关卡设置教程
+ 新建 Data 文件夹放规则类 actor。
+ 类比舞台剧，类似导演后勤角色。
+ 创建蓝图类 Game mode base 规定游戏规则。
### 游戏模式蓝图
+ 类似裁判角色，虚幻引擎指定架构。
+ 在世界场景里设置，可以每关不同。
+ 创建并命名为 GM_InGame。
### 加载游戏规则
+ 在世界设置中指定 game mode。
+ 选择GM_InGame，引擎测试时加载。
+ 保存变更，测试蓝图类变化。
### 蓝图可视化编程
+ 可视化编程直观，非程序员易理解。
+ 有默认事件，begin play 和 event tick。
+ 可进行测试，如显示文字等操作。
### 调试节点介绍
+ print string 节点用于调试。
+ 可设置打印到屏幕或日志等选项。
+ 对初学者更友善，设置安全范围。
### 材质系统设置
+ 新建 materials 文件夹创建材质。
+ 新建材质，命名为M_BasicColor，调整基础颜色，用于临时地面。
+ 可视化编辑材质，编译保存应用。
### 导入人形骨骼结构
+ 添加第三人称模板资产包。
+ 保留骨骼，其他资产可暂不使用。
+ 了解骨骼网格体及相关资产文件。
### 删除多余的资产包内容
+ 删除文件需注意关联关系。
+ 避免强制删除产生问题。
+ 规范命名规则和引用关系很重要。
